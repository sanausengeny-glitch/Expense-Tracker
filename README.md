# Personal Budget & Expense Tracker

## Description

The Personal Budget & Expense Tracker is a web application built as part of the Power Learn Project (PLP) Web Development course. This project will be developed progressively from Week 1 through Week 8, with new features added each week.

The Week 1 version focuses on creating the static structure of the application using HTML5 and CSS3. It provides the basic layout of a budget tracker without any JavaScript functionality.

## Features

- Main page heading and project description
- Add Expense section
- Input fields for:
  - Expense Name
  - Amount
  - Category
- "Your Expenses" section with placeholder text
- External CSS styling
- Use of HTML element, class, and ID selectors

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your preferred web browser.
